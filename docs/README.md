# Portfólio Pessoal - Maghaiver Gomes Ramos

## Descrição

Este é meu **portfólio pessoal**, desenvolvido utilizando **HTML e CSS puro**, com foco em apresentar minhas **hard skills**, projetos realizados e informações de contato. O objetivo é fornecer uma interface clara, responsiva e organizada para que recrutadores ou colegas possam conhecer meu trabalho de forma rápida e objetiva.

## Como Usar / Instalar

1. Clone o repositório:

```bash
git clone https://github.com/SEU-USUARIO/NOME-DO-REPOSITORIO.git
```

2. Navegue até a pasta do projeto:

```bash
cd NOME-DO-REPOSITORIO
```

3. Abra o arquivo `index.html` em um navegador de sua preferência.

## Funcionalidades

* **Seção Bio** com apresentação pessoal e descrição resumida.
* **Hard Skills** apresentando principais tecnologias e ferramentas utilizadas.
* **Projetos** com cards detalhando funcionalidades e links para repositórios no GitHub.
* **Contato** com links para LinkedIn e WhatsApp.
* **Layout responsivo**, compatível com dispositivos móveis e desktops.
* Uso de **Flexbox e CSS Grid** para organização de conteúdo.
